# 앱 1팀

## 프로젝트 소개
- 시간이 애매하거나 귀찮을 때 이용할 수 있는 부경대학교 학생들을 위한 심부름 서비스 앱입니다!


<br>
<br>

  <div align=center><h1> 주요 기능</h1></div>
<ul>
- 부경대 이메일(@pukyong.ac.kr)을 이용하여 부경대 학생만을 위한 심부름 서비스
- 심부름 요청서 작성, 요청서 조회, 심부름 현황 조회, 쪽지 기능

<br>
<br>
</ul>
